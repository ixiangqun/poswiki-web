# 安飞士（Avis）— "我们是第二，所以我们更努力"扭亏为盈

> 行业：汽车租赁
> 定位动作：以"承认第二"为差异化定位，将劣势转为信任来源
> 来源章节：[第4章·心智中的小阶梯](../notes/ch08-心智中的小阶梯.md)

---

## 问题诊断

安飞士在汽车租赁市场连续亏损13年。赫兹（Hertz）牢牢占据第一位置，安飞士找不到突破口，试图通过提升服务质量来赢得竞争——但"我们更好"的主张没有顾客相信。

核心困境：在有强大第一名的品类里，声称自己"更好"，顾客不信；承认第二，反而释放了信任。

---

## 定位方案

文案机构多彼·丹加德·伯恩巴克（Doyle Dane Bernbach）提出极端反直觉的策略：**主动承认第二，将"二号位"转为诚实、努力的心智符号**。

核心口号：**"We're only No.2. We try harder."（我们只是第二，所以我们更努力。）**

逻辑链：
1. 承认第二 → 建立诚实感
2. 诚实感 → 服务承诺可信度提升
3. 服务承诺 → 与赫兹的差异化建立
4. 差异化 → 心智中有明确位置

---

## 执行结果

| 阶段 | 财务结果 |
|------|---------|
| 定位前（连续13年） | 持续亏损 |
| 定位后第1年 | 盈利 **120万美元** |
| 定位后第2年 | 盈利 **260万美元** |
| 定位后第3年 | 盈利 **500万美元** |

市场份额：28% → 36%（定位实施后）

## 理论注解

- [心智阶梯](../concepts/心智阶梯.md) — 安飞士案例说明第二名可以利用梯子位置做文章，而非试图爬到第一
- [差异化原则](../concepts/差异化原则.md) — "我们更努力"是可信、可传播的单一概念，与第一名建立清晰区隔
- [定位理论概述](../concepts/定位理论概述.md) — 定位不改变产品，而是改变顾客心智中的认知框架
