# 约翰·康纳利（John Connally）— 1100万美元只换来1张选票

> 行业：政治
> 定位动作：（反例）高广告投入无法改变已固化的负面心智认知
> 来源章节：[第2章·心智遭受信息轰炸](../notes/ch06-心智遭受信息轰炸.md)

---

## 背景

约翰·康纳利（John B. Connally）是前德克萨斯州州长、前美国财政部长，具有丰富政治经验。1980年美国共和党总统初选时，他试图争夺共和党总统候选人提名，与罗纳德·里根等人竞争。

---

## 问题诊断

康纳利拥有充足的竞选资金，以当时的标准来看属于"大撒把"竞选——募集了大量资金并广泛投入电视广告和竞选活动。

然而，他在选民心智中存在难以改变的**负面定位**：
- 曾因涉嫌受贿被起诉（后虽获无罪，但公众印象已形成）
- 被认为是权谋政客的代表形象
- 与选民的情感连接薄弱

广告和曝光度无法改变这个已固化的心智位置——大量信息轰炸只会强化现有认知，而不会重塑它。

---

## 执行结果

| 指标 | 数据 |
|------|------|
| 竞选投入总额 | **1100万美元** |
| 在初选中获得的代表人票数 | **1张** |
| 单票成本 | **1100万美元/票** |

这成为美国政治史上"广告投入产出比最差"的竞选案例之一，被里斯·特劳特用来说明：**当心智位置已固化，金钱无法改变认知**。

## 理论注解

- [定位理论概述](../concepts/定位理论概述.md) — 定位的核心原则：不要试图改变已经形成的心智认知，而要找到尚未被占据的空位
- [心智阶梯](../concepts/心智阶梯.md) — 一旦某人在顾客心智中被归入特定类别（权谋政客），广告无法将其移出
- [重新定位战略](../concepts/重新定位战略.md) — 康纳利案例说明重新定位的局限：对于极度负面的既有认知，重新定位的难度接近于不可能
