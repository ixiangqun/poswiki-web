# 可宝（Nyquil）— 发现"夜间感冒药"空位成为品类第一

> 行业：药品·感冒药
> 定位动作：在拥挤的感冒药品类中，以"夜间"为定位词开创全新子品类
> 来源章节：[第21章·成功六步曲](../notes/ch25-成功六步曲.md)

---

## 问题诊断

1960-70年代，感冒药市场高度拥挤：康泰克（Contac）、百服宁（Dristan）、NyQuil 的竞争对手们已占据主要货架和顾客心智。正面竞争几乎没有空间。

宝洁旗下 Vicks 面对的问题：如何在已有强势竞争者的感冒药品类中找到可防御的定位？

---

## 定位方案

关键洞察：感冒患者有一个未被满足的特定需求——**夜晚睡前服用**的感冒药。白天的感冒药需要保持清醒，但夜间药物可以含有助眠成分，帮助患者安稳入睡、加速恢复。

**定位词**：**"夜间感冒药"（The Nighttime Cold Medicine）**

策略价值：
1. "夜间"这个词开创了一个全新子品类——之前没有任何品牌声称是"夜间"感冒药
2. 在新子品类中，Nyquil 是第一，不存在竞争
3. 规避了与康泰克、百服宁的正面竞争
4. 产品本身（含助眠成分）支撑了定位主张——可信、有差异

**定位原则应用**：第二步（想拥有什么定位）中"找到心智空位"的教科书案例——Nyquil 没有找更大的空位，而是找了一个窄但可防御的空位："夜间"。

---

## 执行结果

| 结果 | 描述 |
|------|------|
| 市场地位 | 确立为"夜间感冒药"品类领导者 |
| 核心优势 | 成为顾客睡前感冒用药的默认选择 |
| 仿效者困境 | 后来者必须绕开"夜间"这个定位词，Nyquil 的空位防御性极高 |
