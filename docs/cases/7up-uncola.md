# 七喜（7-Up）— "非可乐"关联定位成为全球第三大软饮料

> 行业：消费品·软饮料
> 定位动作：以领导者可乐为参照点，建立"非可乐"的对立定位
> 来源章节：[第4章·心智中的小阶梯](../notes/ch08-心智中的小阶梯.md)

---

## 问题诊断

1960年代，七喜是一款普通的柠檬味汽水，在软饮料市场几乎没有清晰定位。品牌广告投入大量费用描述产品特点（柠檬味、清爽、无咖啡因），但在可口可乐和百事可乐垄断的心智阶梯中，七喜连一个位置都没有。

核心问题：**在一个有超强领导者的品类里，无法靠产品特点获得关注——需要一个与领导者相关的参照点**。

---

## 定位方案

策略转型：**放弃独立描述自身优点，转而以"可乐"为参照，主打"非可乐"（The Uncola）**。

逻辑链：
1. 可乐=心智霸主，拥有巨大的认知资源
2. "非可乐"借用可乐的心智地图，自动定义了七喜的位置
3. 顾客脑中：想买不含咖啡因/不太甜的饮料时，"非可乐"就是选项

核心洞察（里斯·特劳特总结）：**七喜的真正问题不是柠檬味口感，而是可乐在心智中的绝对霸主地位**——正确的战场不是"比口感"，而是"换个类别位置"。

---

## 执行结果

| 指标 | 数据 |
|------|------|
| "非可乐"定位前净销售额 | 约 **8770万美元** |
| "非可乐"定位后净销售额 | **1.90亿美元** |
| 峰值地位 | **全球第三大软饮料品牌** |
| 后续（改变定位策略后） | 份额滑落，最终仅约雪碧的一半 |

## 理论注解

- [心智阶梯](../concepts/心智阶梯.md) — 七喜借助可乐的阶梯位置找到自己的定位，而非试图在可乐阶梯上竞争
- [差异化原则](../concepts/差异化原则.md) — "非可乐"是单一可传播的差异化概念，利用领导者做参照
- [重新定位战略](../concepts/重新定位战略.md) — 不正面攻击可乐，而是将"不选可乐"的场景转化为选七喜的理由
- [品牌延伸陷阱](../concepts/品牌延伸陷阱.md) — 后期改变定位策略导致份额下滑，是"无法坚持定位"的反面教材
