# 王老吉（加多宝）·案例

> 来源：《定位》《商战》附录A·定位思想应用
> 行业：消费品·饮料
> 相关概念：[../concepts/差异化原则](../concepts/差异化原则.md) · [../concepts/品类战略](../concepts/品类战略.md)

---

## 背景

王老吉凉茶在"怕上火"定位概念确立之前，年销售额长期徘徊在1亿元左右，局限于广东市场，未能走向全国。

---

## 问题

凉茶品类认知度低，消费者不理解为何需要凉茶，品牌定位模糊，产品缺乏明确的使用场景。

---

## 方案

2002年，借助特劳特（中国）公司确立"**怕上火，喝王老吉**"定位概念——将"预防上火"作为凉茶品类的核心价值主张，绑定明确场景（吃火锅、烧烤等"容易上火"的场合），由广东成功走向全国。

---

## 结果与启示

| 指标 | 数据 |
|------|------|
| 定位前年销售额 | 约1亿元（徘徊多年） |
| 2008年销售额 | **120亿元** |
| 增长倍数 | 约120倍 |
| 对标参照 | 2008年超越可口可乐在中国的销售额 |

- **启示**：一个词语（"怕上火"）开创/重新定义了一个品类，使其从区域性饮料成为全国性品类领导者
- 定位的场景化（火锅/烧烤场合）是关键执行：把"怕上火"转化为可感知的具体需求
