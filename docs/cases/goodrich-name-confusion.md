# 固特里奇（Goodrich）— 56%顾客把创新归功于竞争对手

> 行业：汽车·轮胎
> 定位动作：（反例）品牌名与强势竞争对手高度相似，导致创新成果被误归
> 来源章节：[第9章·名字的威力](../notes/ch13-名字的威力.md)

---

## 问题诊断

固特里奇（B.F. Goodrich）是美国老牌轮胎公司，拥有真实的技术创新能力。问题在于：竞争对手固特异（Goodyear）比固特里奇更出名，而两个名字高度相似——

- **Goodrich（固特里奇）**
- **Goodyear（固特异）**

两者都含有"Good"前缀，读音和拼写近似，顾客极易混淆。

---

## 核心数据

调研测试结果：固特里奇发明了**钢带子午线轮胎**（一项重要技术创新），但当顾客被问及"这项技术是哪家公司发明的"时：

| 回答 | 比例 |
|------|------|
| 固特异（Goodyear）——**竞争对手** | **56%** |
| 固特里奇（实际发明者） | 少数 |

超过半数的顾客将固特里奇的技术突破记在了固特异的账上。

**结果**：固特里奇花费研发资源开发的创新，大部分品牌价值被竞争对手"免费获得"。

---

## 理论注解

- [命名战略](../concepts/命名战略.md) — "糟糕名字六种类型"之一：品牌名与强势竞争对手过于相似。相似的名字使顾客在做品牌归因时系统性出错
- [差异化原则](../concepts/差异化原则.md) — 差异化不仅是产品特性的差异，也包括命名的差异——无法被独特识别的品牌，差异化等于零
- [定位理论概述](../concepts/定位理论概述.md) — 认知即现实：固特里奇的技术实力再强，若顾客认为那是固特异的成就，商业价值就流向了竞争者
