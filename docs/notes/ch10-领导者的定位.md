## 核心主张

1. **第一品牌的份额法则**：第一个进入心智的品牌长期市场份额通常是第二名的2倍、第三名的4倍，且这种比例关系不会轻易改变
2. **"正宗货"战略**：领导者应强化"我们是原创"而非"我们是第一"——"正宗货"让所有竞争者都成了仿冒品
3. **全面拦截**：一旦发现有潜力的新品类/新产品，立即跟进，抢在对手扎根心智之前狙击
4. **多品牌=单一定位**：宝洁每个品类独立品牌，而非品牌延伸——这是正确的拦截工具
5. **企业实力来自产品定位，而非企业规模**：施乐=复印机，一旦脱离这个定位，它就从零开始；可口可乐公司的实力是可口可乐（产品）在心智中地位的反映
6. **成为第一很难，保持第一要简单得多**——领导者短期内坚不可摧，仅凭势能就能维持

## 关键数据

| 指标 | 数值 |
|------|------|
| 第一品牌 vs 第二品牌市场份额 | 通常为2:1（可口可乐6:百事可乐4） |
| 第一品牌 vs 第三品牌 | 通常为4:1 |
| 1923年25个品类领导品牌 | 77年后仍有22个保持领导地位 |
| 通用汽车1978年利润率 | 市场份额49%，净利润率6.1% vs 美国汽车公司份额2%，利润率0.4% |
| 佳洁士延伸50款产品 | 最终让出领导地位给高露洁 |
| 施乐进军电脑 | 20年亏损数十亿，台式激光打印机机会拱手让给惠普 |

## 关键案例

- **强生/泰诺 vs 达特利**：达特利发起价格战，强生立即降价拦截，速度之快连达特利的降价广告都未来得及推出——拦截的关键是时机，必须在对手扎根之前出手
- **宝洁命名逻辑**：洗衣粉用"汰渍"、洗洁精用"Cascade"，而非"汰渍洗洁精"——每个品类独立品牌，互不干扰，吉列多品牌共占60%市场份额
- **丰田雷克萨斯**：豪华车独立品牌，而非"超级版丰田"——多品牌拦截的教科书案例
- **柯达/3M错失施乐**：两家公司拒绝购买静电复印技术，哈洛伊德公司冒险买下，最终成为价值50亿美元的施乐
- **喜力失守**：放弃"美国进口啤酒第一品牌"说法，最终将领导地位拱手让给特级科罗娜

## 金句

> "企业的实力来自产品的实力，即产品在潜在顾客心智中所占据的定位。"

> "成为第一很难，保持第一要简单得多。"

> "'正宗货'就像初恋，在潜在顾客的心智中永远占据着一个特殊的位置。"
