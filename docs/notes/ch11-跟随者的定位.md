# 第7章：跟随者的定位

## 核心主张

1. **"寻找空位"**（法语：cherchez le creneau）——跟随者的核心战略不是做得比领导者更好，而是在潜在顾客心智中找到尚未被占据的空位
2. **空位无须惊天动地**——不需要有明显的顾客利益点，只需是第一个占据那个心智位置（劳力士=奢华手表、Orville Redenbacher=高价爆米花、米狮龙=国产高端啤酒）
3. **工厂空位陷阱**：企业常犯的错误是填补工厂的生产空缺，而非顾客心智中的空位——从内部逻辑出发定位，而非从外部心智出发
4. **技术陷阱**：产品技术上是第一，不等于心智中是第一——Frost 8/80是"第一款白威士忌"，但心智中威士忌是棕色的
5. **满足所有人需求的陷阱**：竞争激烈时，不聚焦就等于失败；必须主动放弃某些市场才能赢得某个市场

---

## 六类空位类型

| 空位类型 | 案例 | 定位逻辑 |
|---------|------|---------|
| **尺寸空位** | 大众甲壳虫"想想小车" | 所有人都往大造，小就是空位 |
| **高价空位** | 米狮龙、皇家芝华士、劳力士 | 第一个建立高价定位，高价本身即差异化 |
| **低价空位** | Qwip传真机（$29 vs 施乐$45） | 新品类中顾客愿意以低价尝鲜 |
| **性别空位** | 万宝路（男子汉）、维珍妮（女性） | 所有品牌都往一个性别走，另一个性别就是空位 |
| **年龄空位** | 巨力多（老年人）、Aim牙膏（孩子） | 现有品牌都面向主流年龄段 |
| **渠道空位** | 蛋袜L'eggs（超市卖袜子） | 第一个在非传统渠道销售 |

---

## 关键案例

- **大众甲壳虫**："想想小车"——当所有汽车品牌争相造大车时，小型车是空位；该广告同时说明了定位（小）并挑战了"越大越好"的假设
- **万宝路**：第一个建立全国性男子汉香烟定位，10年内从全国第五升至第一；跟风品牌Luke晚了20年，终告失败
- **埃德塞尔**：福特为填补林肯-水星-福特之间的"工厂空缺"而生，但心智中中端汽车早已饱和——工厂空位陷阱的经典教材
- **Frost 8/80白威士忌**：技术上确实是"第一款白威士忌"，但心智中威士忌是棕色的；两年内亏损数百万，销量仅达预期1/3

## 金句

> "找出空位，然后填上。"

> "要想在当今的竞争环境里取胜，你必须走出去交朋友，切取一个细分的利基市场，尽管你会因此失去其他市场。"

> "不要和潜在顾客玩文字游戏。广告并非辩论，而是为了'诱惑'消费者。"
