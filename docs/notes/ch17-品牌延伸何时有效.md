# 第13章：品牌延伸何时有效

> 来源：《定位》第13章
> 核心概念：[../concepts/品牌延伸陷阱](../concepts/品牌延伸陷阱.md) · [../concepts/命名战略](../concepts/命名战略.md)

---

## 核心主张

1. **短期有利、长期不利**：品牌延伸名前6个月渠道铺货顺利、数据好看；6个月后无重复订单，慢性病发作
2. **三幕悲剧模型**：①大成功（找到空位占据定位）→ ②品牌延伸（用同一名字进攻新市场）→ ③梦想幻灭（多款产品总销量不如只聚焦一款）
3. **品牌延伸5条可行原则**：满足特定条件时品牌延伸是可接受的（小销量、无竞争、无广告等）
4. **两大实操测试工具**：购物清单测试法（发现顾客默认联想的品类）；调酒师测试法（检验延伸名是否带来点单混乱）
5. **橡皮筋原理**：品牌名可以拉伸，但延伸越长越脆弱——不是不能拉，是拉断了才知道太长

---

## 关键案例

**三幕悲剧经典案例：**
- **大众甲壳虫**：1965年一款车型占美国进口车市场67%；推出多车型品牌延伸后，1979年跌至12%，1993年不足3%；重新推出甲壳虫，销量立即回升
- **凯迪拉克赛威/西马龙**：为应对油价推出小型凯迪拉克，短期卖出销量，长期动摇"大型豪华轿车"定位；西马龙最终失败，卡特拉（"灵活自如的凯迪拉克"）同样告败
- **帕卡德**：20世纪30年代推出低价车型Clipper，短期大卖；最终摧毁"尊贵"定位，1954年被并购，1958年彻底倒闭
- **沃尔沃**：从"可靠"→ 同时宣称豪华、安全、驾驶乐趣、旅行版 → 销量持续下滑；重新聚焦"安全"后销量回升至40万辆

**两大测试工具：**
- **购物清单测试法**：清单写"拜耳"，爱人买回来的必然是阿司匹林，而非"拜耳非阿司匹林止痛药"——这正是延伸名被心智忽略的证明
- **调酒师测试法**：点"顺风加冰"得到威士忌，没问题；点"顺风12加冰"，调酒师可能没听清"12"——品牌延伸名在公开场合制造发音尴尬

## 品牌延伸的5条可行判断原则

| 条件 | 不应使用公司名（→独立品牌） | 应使用公司名（→品牌延伸可接受） |
|------|--------------------------|-------------------------------|
| 预期销量 | 销售潜力大 | 销量小 |
| 竞争格局 | 竞争激烈 | 无竞争 |
| 广告资源 | 有大广告预算 | 预算很小 |
| 产品凸显性 | 突破型创新产品 | 货品化产品（如化学品） |
| 分销渠道 | 在货架上销售 | 由销售代表推销 |

## 金句

> "品牌名好比橡皮筋，可以拉伸，但有临界点。品牌名延伸得越长，就越脆弱。"

> "品牌延伸是一种慢性病，潜伏多年才会爆发，过程漫长而不易察觉。"

> "开发一个新概念或新产品，为它找到一个新定位，并配以一个新名字——这是进军新领域的正确方法。"
