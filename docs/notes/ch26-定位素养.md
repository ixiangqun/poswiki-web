# 第22章：定位素养

> 来源：《定位》第22章
> 核心概念：[../concepts/定位理论概述](../concepts/定位理论概述.md)

---

## 核心主张

1. **词语是空的容器**：词语没有含义，含义在于人们使用它的方式——名字是可以更换的容器，不应被历史和词典束缚
2. **人类不完全理智**：大多数人先有看法再找证据；能够激发正面认知的词语具有强大威力
3. **应对变化的正确方式是守住定位**：不是随变化而调整，而是找到基础业务并坚守；盲目跟风变化的公司（RCA进入电脑、美国橡胶进入非橡胶）均告失败
4. **定位是外部导向的**：真正的营销人员把注意力放在竞争对手上，而非自我激励

---

## 定位素养的七个维度

| 素养 | 核心内容 |
|------|---------|
| **远见** | 对企业未来5-10年做决策，不为每次变化调整方向 |
| **勇气** | 在局势未定时倾力投入；好时公司因不打广告失去领导地位 |
| **客观** | 消除自我主义，用外部视角看产品；需要搭档进行思想碰撞 |
| **简单** | 只有显而易见的定位概念才可行；"解决问题的答案将是简单的" |
| **敏锐** | 成功定位在中间地带附近，而非两极；平衡独一无二与广泛吸引力 |
| **耐心** | 按地域/人群/年龄/渠道逐步展开，而非全面铺开 |
| **全球视角** | 利用一国定位开辟下一国市场；名字需要具备全球可用性 |

---

## 内部导向 vs 外部导向

| 内部导向型 | 外部导向型 |
|-----------|-----------|
| 相信决心和努力就能成功 | 把注意力集中在竞争对手上 |
| 相信"我们的员工最优秀" | 像将军巡视战场那样扫描市场 |
| 从公司内部找定位 | 到外部（顾客心智中）找定位 |
| 结果：自我激励、方向模糊 | 结果：发现竞争对手的弱点加以利用 |

---

## 金句

> "在心智中拥有一个定位，如同拥有高价值的不动产。你一旦放弃，就会发现不可能再拿回来。"

> "定位的第一法则：要想赢得心智之战，就不能和已经在顾客心智中牢牢占据强有力位置的企业正面交锋。"

> "如果不是领导者，那就开创一个能够让你成为领导者的新品类。"
