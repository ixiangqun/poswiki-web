# 第15章：国家定位——比利时

## 核心主张

1. **最有效的战略不是为航空公司定位，而是为目的地（国家）定位**：比利时航空公司真正的问题是比利时在旅行者心智中没有位置
2. **第三方可信状加持**：《米其林指南》的三星评级比自我宣传更有说服力——客观外部背书是定位的有力工具
3. **关联法则**：将未知事物与心智中已知的强势定位挂钩（"五个阿姆斯特丹"而非从零开始）
4. **简单原则**：加入更多城市只会增加混淆；定位需要极度简化信息，一个核心概念即可
5. **执行失败的教训**：再好的定位战略，若管理层换届或内部政治干扰，不能坚持执行，定位就会消失

---

## 关键案例

**比利时三星城市战略**：
- 洞察：《米其林指南》比荷卢版列出6个三星城市，5个在比利时（布鲁日、根特、安特卫普、布鲁塞尔、图尔奈），荷兰只有阿姆斯特丹1个
- 定位主题："在美丽的比利时，有五个阿姆斯特丹"
- 效果：广告吸引大量旅游咨询，荷兰旅游局局长致电抗议
- 失败原因：比利时航空公司管理层换届，新管理层回到"欧洲门户"旧战略，定位未能持续执行

## 金句

> "无论你推销的是可乐、企业还是国家，若没有进入心智，就没有业务。"

> "需要牢记的教训是：定位需要极度简化信息。混淆是大敌，简单是制胜法宝。"

> "一个成功的定位项目需要负责人长期全力执行。"
