# 商战·第9章：侧翼战

> 来源：《商战》里斯·特劳特
> 章节：第9章

---

## 核心主张

**侧翼战三原则，是《商战》中创新性最强的战略形式：**

1. **最佳的侧翼战应该在无争地带进行。** 不是攻击已有市场，而是在竞争对手未占领的细分维度（价格、功能、地理、人群）开辟新阵地。侧翼战在本质上是品类战略的军事化表达。
2. **战术奇袭应该成为作战计划中最重要的一环。** 一旦侧翼战动作被领导者提前感知，领导者有足够资源快速复制，侧翼战窗口关闭。速度和出其不意是侧翼战的核心护城河。
3. **追击与扩大战果，和进攻本身同等重要。** 许多侧翼战成功发起后，企业因保守心理而放弃追击，结果让领导者有时间追上来。侧翼战成功后必须快速加大投入，扩大战果。

**侧翼战 vs 进攻战的区别**：进攻战打在竞争对手的正面（找其强势中的弱点），侧翼战打在竞争对手的侧面（找其未防守的新维度）。

## 关键案例

- **七喜"非可乐"**（→[cases/7up-uncola](cases/7up-uncola.md)）：在可乐品类之外开辟"非可乐"细分。净销售额8770万→1.9亿美元。无争地带 + 出其不意 + 持续追击三要素齐备
- **米勒淡啤**（→[cases/miller-lite](cases/miller-lite.md)）：在高卡路里啤酒市场之外，开辟"淡啤"新品类。初期极为成功，但后来因未能保护"Lite"商标（未持续追击防守），被百威 Light 压制
- **万宝路**（→[cases/marlboro-cowboy](cases/marlboro-cowboy.md)）：在女性定位香烟市场之外，开辟"男性/牛仔"定位细分。10年内从全国第五升至第一
