# 序一：定位——第三次生产力革命

> 来源：《定位》《商战》共同序言（邓德隆，里斯伙伴中国公司）
> 核心概念：[../concepts/定位理论概述](../concepts/定位理论概述.md) · [../concepts/品类战略](../concepts/品类战略.md) · [../concepts/差异化原则](../concepts/差异化原则.md)

---

## 核心主张

1. **定位是第三次生产力革命**：泰勒科学管理（体力工作者）→ 德鲁克组织管理（知识工作者）→ 里斯特劳特定位理论（心智工作者）
2. **竞争重心已从市场转移到心智**：工厂时代→渠道时代→心智时代，经济权力随之从工厂→渠道→顾客心智
3. **A模式（聚焦）系统性优于J模式（多元）**：有量化数据支撑，不是观点分歧而是事实差距
4. **定位理论是A模式背后的理论**，是美国企业竞争力的重要来源

## 定位理论四个发展阶段

| 年代 | 阶段 | 核心贡献 |
|------|------|---------|
| 1970s | 定位诞生 | 营销竞争的终极战场是心智，而非工厂或市场 |
| 1980s | 营销战（《商战》） | 防御/进攻/侧翼/游击四种战略形式 |
| 1990s | 聚焦 | 企业竞争力=激光式聚焦，而非太阳般散射 |
| 2000s | 开创新品类 | 消费者"以品类思考，以品牌表达"；开创并主导品类是企业唯一目标 |

---

## 中国A模式实证

- **格力**：聚焦空调，利润率5%-6%（vs 海尔多元化后利润率约1%）
- **长城汽车**：聚焦SUV后哈弗夺冠，逆市增长
- **任天堂（日本）**：聚焦游戏机，利润率22%（vs 索尼多元化1.7%）
- **诺基亚（欧洲）**：砍掉橡胶/造纸/彩电，聚焦手机，10年超越百年企业西门子成欧洲第一
