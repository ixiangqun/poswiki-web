## 核心主张

1. **媒体暴增 + 产品暴增 + 广告暴增**三重叠加，传播渠道严重堵塞——多即少，传播量越大，信息损失率越高
2. 心智容量有限（"不到一升的容器"），科学上已证实"感觉超负荷"现象：超过临界点，大脑会一片空白
3. 在传播过度的社会里，**没有传播有时反而有利**——第一印象一旦建立，几乎无法通过广告改变
4. 穿越传播堵塞的唯一方法：**极度简化，集中在有限信息上**（雪佛兰式答案）

## 关键案例

- **雪佛兰**：9款车型，每天广告投入35.6万美元，但品牌形象模糊，反落后于福特——信息越多越分散，适得其反
- **约翰·康纳利**：总统初选花费1100万美元，仅获1张选票——已有负面认知"独断专行"无法靠广告覆盖
- **杰里·布朗**：大量媒体报道、4本传记，公众印象仍只停留在4件事上——传播量≠心智占有量
