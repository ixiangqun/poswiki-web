# 商战·第16章：商业将领

> 来源：《商战》里斯·特劳特
> 章节：第16章

---

## 核心主张

**优秀的商业将领具备五个核心特质，缺一则降级为平庸管理者。** 里斯·特劳特区分了"领导者"（鼓舞士气）和"将领"（制定并执行战略）——两者是不同的角色。企业在需要将领时选择了领导者，在需要领导者时选择了将领，是系统性配置错误。

**五大特质**：
1. **灵活**：调整战略以适应形势，而非让形势适应战略
2. **决断力**：开放收集信息，果断关闭做决策；不固执于已有战略
3. **魄力**：时机成熟时果断倾力而出；地位越高越容易失去魄力
4. **通晓事实**：战略从细节出发，自下而上；不忽视战术细节
5. **通晓原则**：熟练到不需要思考原则，能将全部注意力放在竞争对手上

**分权管理的危害**：分权管理使战略制定责任从最高层下移，导致战略失效。巴顿的第三集团军105名军官中只有1名战略规划人员。

---

## 关键案例

- **杰克·韦尔奇（通用电气）**：将领特质-灵活。任何不在行业数一数二的业务，整顿、出售或终止，从不固守
- **李·艾柯卡（克莱斯勒）**：将领特质-决断力 + 魄力。接手濒临破产的克莱斯勒，以一系列侧翼战（小型车、民粹营销）成功转型
- **约翰·里德（花旗银行）**：将领特质-通晓事实。率先推出ATM机，通晓零售银行战术细节后做出的战略决策
- **汉堡王杰弗里·坎贝尔**：将领特质-魄力。采用"烤制非油炸"战略，果断向麦当劳发起进攻
- **惠普卡莉·菲奥莉娜**：反例。出色的领导者（鼓舞士气），糟糕的将领（战略上合并康柏毫无道理）
