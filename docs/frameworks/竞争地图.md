# 竞争地图框架

> **用途**：可视化市场竞争格局，识别定位空白区。

---

## 坐标轴选择原则

- X轴、Y轴必须是**客户真正在意的权衡维度**（非内部KPI）
- 两轴之间要有**合理的对立张力**（例：价格 vs 品质；功能 vs 情感）
- 避免使用"高端/低端"这类模糊词

---

## 标准竞争地图

```
       高 [Y轴属性]
        |
        |   竞品A ●        ● 竞品B
        |
        |              ★ 定位空白区
        |
        |   竞品C ●
        |
低 [Y轴属性] ——————————————————→ 高 [X轴属性]
                    ↑
               本品牌当前位置 ●
```

---

## 解读规则

1. **聚集区**：多个竞品重叠 → 红海，需要差异化逃离
2. **空白区**：无竞品覆盖 → 候选定位，但需验证"空白是否有需求"
3. **本品牌位置**：如与竞品重叠，定位无效；如在空白区，验证可信度
